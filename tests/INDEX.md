## Running Tests

### Run All Tests
```bash
npm test
```

### Run Specific Test Category
```bash
npm test -- tests/evasion/
npm test -- tests/deployment/
npm test -- tests/integration/
```

### Run Individual Test
```bash
npm test -- tests/comprehensive-integration-test.js
```

### Load Testing
```bash
node tests/load-test-v12.js
node tests/load-testing-analysis.js
```

### Performance Profiling
```bash
node tests/comprehensive-performance-analysis.js
node tests/performance-profiler-advanced.js
```
