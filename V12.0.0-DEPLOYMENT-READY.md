
- Error rate > 5% for > 5 minutes
- Memory growth > 5MB/hour sustained

**Rollback Duration:** < 2 minutes

**Process:**
```bash
# 1. Stop v12.0.0

# 2. Restore v11.3.0

# 4. Start v11.3.0
```
