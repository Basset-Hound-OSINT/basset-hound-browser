DEBUGGING

Priority Queue Issues:
  - Check: priorityMap has all command types
  - Low-priority starving? Increase fairnessConfig.lowPriorityProcessInterval
  - Queue breakdown in status shows distribution

DOM Cache Issues:
  - Hit rate too low? Check URL patterns, increase TTL
  - Memory growing? Reduce maxCacheSize or enable compression
  - Stale data? Use forceFresh param, cache TTL is 5s default

Screenshot Issues:
  - Latency not improving? Check buffer contention
  - All buffers busy? Reduce concurrent screenshot requests
  - Quality issues? Adjust webpQuality parameter (85 default)
