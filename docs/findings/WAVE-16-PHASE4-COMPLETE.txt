DEPLOYMENT NOTES
===============================================================================

Dependencies:
- No new npm packages required (uses Node.js built-in modules)
- Optional: Redis client for distributed caching
- Optional: Elasticsearch/Opensearch client for search

Configuration:
- Cache manager: Configurable memory limits, TTL, eviction policy
- Query cache: Per-query TTL and dependency configuration
- Analytics: Retention policies and aggregation intervals
- Search: Index shard/replica configuration
- Validation: Custom validator registration

Environment Variables:
- CACHE_MAX_MEMORY: Maximum cache memory (default: 100MB)
- CACHE_EVICTION_POLICY: LRU or LFU (default: LRU)
- ANALYTICS_RETENTION_DAYS: Default retention (default: 30)
- SEARCH_BULK_SIZE: Bulk indexing batch size (default: 100)

Data Persistence:
- Disk cache path: Configurable, defaults to .cache
- Analytics data: In-memory, exported to file on demand
- Search indices: Elasticsearch/Opensearch managed
- Database: Through repository via provided data store
