NEW VULNERABILITIES (7)
================================================================================

1. CVE-W14-NEW-001: Insecure Proxy ID Generation [CRITICAL CVSS 8.2]
   - Component: /src/proxy/residential-proxy-manager.js
   - Impact: Session hijacking via proxy ID enumeration
   - Fix Effort: 30 minutes
   - Root Cause: Using Date.now() + Math.random() instead of crypto.randomBytes()
   - Status: ⚠️ UNFIXED - Requires immediate remediation

2. CVE-W14-NEW-002: Math.random() in Proxy Rotation [CRITICAL CVSS 7.8]
   - Component: /src/proxy/residential-proxy-manager.js
   - Impact: Fingerprinting via predictable proxy selection
   - Fix Effort: 30 minutes
   - Root Cause: Using Math.random() for security-sensitive operation
   - Status: ⚠️ UNFIXED - Requires immediate remediation

3. CVE-W14-NEW-003: No Timeout on JSDOM Parsing [CRITICAL CVSS 7.5]
   - Component: /src/monitoring/change-detector.js
   - Impact: Denial of Service via pathological HTML
   - Fix Effort: 45 minutes
   - Root Cause: No timeout wrapper or size limits on JSDOM parsing
   - Status: ⚠️ UNFIXED - Requires immediate remediation

4. CVE-W14-NEW-004: Unvalidated Proxy Reputation [CRITICAL CVSS 7.2]
   - Component: /src/proxy/proxy-intelligence.js
   - Impact: Session hijacking via reputation spoofing
   - Fix Effort: 2 hours
   - Root Cause: Result parameter not verified for authenticity
   - Status: ⚠️ UNFIXED - Requires immediate remediation

5. CVE-W14-NEW-005: Weak UUID Generation [HIGH CVSS 7.1]
   - Component: /src/export/platforms/stix-export.js
   - Impact: UUID collision and enumeration attacks
   - Fix Effort: 30 minutes
   - Root Cause: Math.random() used for UUID segment generation
   - Status: ⚠️ UNFIXED

6. CVE-W14-NEW-006: Weak Session Token Generation [MEDIUM CVSS 6.5]
   - Component: /src/evasion/multi-layer-coordinator.js (and others)
   - Impact: Token prediction and enumeration
   - Fix Effort: 30 minutes
   - Root Cause: Combining predictable timestamp with weak randomness
   - Status: ⚠️ UNFIXED

7. CVE-W14-NEW-007: No Snapshot Size Validation [MEDIUM CVSS 6.3]
   - Component: /src/monitoring/monitoring-service.js, /src/sessions/session-persistence.js
   - Impact: Memory exhaustion DoS
   - Fix Effort: 45 minutes
   - Root Cause: No maximum size enforcement on JSON snapshots
   - Status: ⚠️ UNFIXED
