USAGE EXAMPLES
================================================================================

Basic Onboarding Flow:
```javascript
const coordinator = new OnboardingCoordinator({ userId: 'user-1' });
await coordinator.initialize();

// Register steps
coordinator.registerStep('step-1', { name: 'Welcome', handler: () => {} });
coordinator.registerStep('step-2', { name: 'Setup', prerequisites: ['step-1'] });

// Navigate
await coordinator.goToStep('step-1');
await coordinator.completeStep('step-1', { data: 'value' });
await coordinator.nextStep();

// Generate certificate on completion
const cert = await coordinator.generateCertificate();
```

Setup Wizard:
```javascript
const wizard = new SetupWizard(coordinator);

// Advance through steps with validation
const result = await wizard.nextStep();
await wizard.saveStepData('welcome-setup', { fullName: 'John' });

// Complete setup
const completion = await wizard.completeSetup();
```

Certification:
```javascript
const certification = new CertificationEngine({ userId: 'user-1' });

// Start exam
const session = await certification.startExam('exam-basic-monitoring');

// Submit answers
certification.submitAnswer('exam-basic-monitoring', 'q-1', 0);

// Grade exam
const result = await certification.submitExam('exam-basic-monitoring', answers);
if (result.result.passed) {
  const cert = certification.verifyCertificate(result.result.certificateId);
}
```
