================================================================================
TROUBLESHOOTING
================================================================================

Server Not Running
  Error: "ECONNREFUSED 127.0.0.1:8765"
  Solution: Start WebSocket server on port 8765
  Command: node websocket/server.js

Memory Growth Too High
  Issue: >5 MB/hour detected
  Cause: Possible memory leak
  Action: Review heap snapshots, enable GC logging
  Debug: node --expose-gc tests/stability/24-hour-session.test.js

Latency Degradation
  Issue: P99 latency increases over time
  Cause: Possible GC pauses or resource exhaustion
  Action: Monitor CPU/memory, check for blocking operations
  Debug: Use Node Inspector (--inspect) for profiling

Connection Leaks
  Issue: Open connections not closing
  Cause: Improper cleanup in error handlers
  Action: Review connection close handlers, add explicit cleanup
  Debug: Check 'netstat -an' for open sockets

Test Timeout
  Issue: Test runs longer than expected
  Cause: Slow network, high latency, hung process
  Action: Increase timeout, check network conditions
  Command: timeout 3600 node quick-validation.js (increase timeout)
