✅ Confidence Scoring System
   - Base accuracy from signature (0.5-1.0)
   - Boost for multiple detection methods (+0.05 per additional method)
   - Final confidence calculation (0-1 scale)

✅ Caching System
   - Optional result caching (enabled by default)
   - LRU cache with configurable TTL
   - Cache invalidation on demand

✅ Error Handling
   - Graceful handling of missing/null inputs
   - No-crash on malformed HTML
   - Comprehensive error logging
   - Result formatting even on errors

Configuration Options:
- minConfidence: Minimum confidence threshold (default 0.50)
- maxResults: Maximum number of results (default 100)
- enableVersionDetection: Version extraction (default true)
- cacheResults: Result caching (default true)
- cacheTTL: Cache time-to-live in ms (default 5 minutes)

Methods:
- detect(pageData): Main detection method
  Input: { html, headers, url }
  Output: { success, technologies[], totalDetected, scanTimeMs, timestamp }

- clearCache(): Clear cached results
