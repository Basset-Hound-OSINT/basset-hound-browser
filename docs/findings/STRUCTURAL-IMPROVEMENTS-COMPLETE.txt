PERFORMANCE CHARACTERISTICS
================================================================================

LRU CACHE:
  - Set: O(1) amortized
  - Get: O(1) amortized
  - Evict: O(1) amortized
  - TTL cleanup: O(n) on evictExpired() call
  - Memory: O(n) where n = maxSize

DEPENDENCY CONTAINER:
  - Register: O(1)
  - Get (singleton): O(1) after first call
  - Get (transient): O(1) factory call + instance creation time
  - Validation: O(services * dependencies)
  - Memory: O(services)

SERVICES:
  - Creation: O(1)
  - Operations: O(1) to O(n) depending on operation
  - Stats: O(1) return time
  - Memory: O(items tracked)
