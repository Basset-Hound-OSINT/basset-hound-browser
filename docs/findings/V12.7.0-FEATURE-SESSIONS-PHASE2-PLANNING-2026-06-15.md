#### Task 4-2: Memory Leak Detection

**File:** `tests/integration/memory-leak-detection.test.js` (new)

**Purpose:** Identify and validate absence of memory leaks

**Profiling Strategy:**

1. **Baseline Measurement**
   - Measure initial heap size
   - Force garbage collection
   - Record baseline memory

2. **Load Generation**
   - Execute 1000 commands in sequence
   - Save/restore state 50 times
   - Create/destroy 10 sessions

3. **Post-Load Measurement**
   - Force garbage collection
   - Measure final heap size
   - Calculate growth rate

4. **Leak Analysis**
   - If growth > 10 MB/1000 commands: Potential leak
   - Profile heap snapshots to find source
   - Validate no detached DOM nodes
   - Check for circular references

**Implementation:**
```javascript
class MemoryLeakDetector {
  async detectLeaks() {
    const baseline = this.captureMemory()
    
    // Generate load
    for (let i = 0; i < 1000; i++) {
      await this.executeCommand()
    }
    
    // Force GC and measure
    global.gc()
    const final = this.captureMemory()
    
    // Analyze
    const growth = final.heapUsed - baseline.heapUsed
    const growthPerCommand = growth / 1000
    
    if (growthPerCommand > 10 * 1024) { // 10KB per command
      // Likely leak detected
      await this.analyzeHeapSnapshot()
    }
    
    return {
      leaked: growthPerCommand > threshold,
      growth: growth,
      growthPerCommand: growthPerCommand
    }
  }
}
```
