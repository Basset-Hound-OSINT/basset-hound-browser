INTEGRATION INSTRUCTIONS
================================================================================

STEP 1: Install Dependencies
  npm install ajv ajv-formats

STEP 2: Generate HMAC Secret
  export HMAC_SECRET=$(node -e "console.log(require('crypto').randomBytes(32).toString('hex'))")

STEP 3: Integrate into websocket/server.js
  - Import all 6 modules
  - Initialize in constructor
  - Add 5-step security pipeline:
    1. Verify message signature (HMAC)
    2. Check command authorization
    3. Validate input parameters
    4. Execute command with protections
    5. Sanitize and sign response

STEP 4: Set Environment Variables
  NODE_ENV=production
  BASSET_WS_REQUIRE_WSS=true
  BASSET_WS_ALLOWED_ORIGINS=api.yourdomain.com

STEP 5: Run Tests
  npm test -- tests/security/

STEP 6: Deploy to Staging
  npm run build
  npm run deploy:staging

  - Test input validation bypass

  - Test JS sandbox escape
  - Test HMAC tampering
  - Test data leakage
