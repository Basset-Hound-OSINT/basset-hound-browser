================================================================================
TROUBLESHOOTING
================================================================================

Connection Refused?
  1. Check Electron is running: ps aux | grep electron
  2. Wait 5+ seconds after npm start
  3. Check port: lsof -i :8765

Timeout Error?
  1. Increase TIMEOUT in script
  2. Check network connectivity
  3. Verify selector matches page

See GOOGLE-SEARCH-FETCH-SETUP.md for detailed troubleshooting
