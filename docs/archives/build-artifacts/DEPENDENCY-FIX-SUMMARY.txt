================================================================================
RATIONALE FOR SPECTRON REMOVAL
================================================================================

Why Remove Instead of Update?

1. Unmaintained: Last update ~2021 (5 years old)
2. Legacy Code: Uses old webdriverio v4 (current is v8+)
3. Not Core: Only used in tests - not in production code
4. Better Alternative: Playwright already handles Electron testing better
5. Cleaner Tree: Removing entire branch saves 211 packages
6. No Loss: All test coverage migrated to Playwright

Why Not Update to Spectron 19?

1. Would introduce different testing API (breaking change)
2. Playwright already tested and working (no need to add complexity)
3. Playwright is the industry standard for Electron testing
4. Less code to maintain (one framework instead of two)
