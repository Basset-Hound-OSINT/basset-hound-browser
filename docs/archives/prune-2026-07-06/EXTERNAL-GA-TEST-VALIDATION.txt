RUNNING THE TEST SUITE
═══════════════════════════════════════════════════════════════════════════════

Prerequisites:
  - Node.js installed
  - Jest test framework
  - Basset Hound Browser running

Installation:
  npm install

Start Browser (Terminal 1):
  npm start

Run Tests (Terminal 2):
  jest tests/external-ga-readiness.test.js --runInBand

Run Specific Test Suite:
  jest tests/external-ga-readiness.test.js \
    --testNamePattern="API Self-Documentation"

Run with Coverage:
  jest tests/external-ga-readiness.test.js --coverage

Run Verbose:
  jest tests/external-ga-readiness.test.js --verbose

Configuration (Environment Variables):
  WS_PROTOCOL=ws (or wss)
  WS_HOST=localhost
  WS_PORT=8765
  HTTP_PROTOCOL=http (or https)
  HTTP_HOST=localhost
  HTTP_PORT=8765
  TLS_ENABLED=false
  TLS_REJECT_UNAUTHORIZED=true
