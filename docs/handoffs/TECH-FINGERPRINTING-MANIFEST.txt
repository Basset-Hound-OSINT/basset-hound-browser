KNOWN LIMITATIONS & WORKAROUNDS
================================================================================

1. HTTP Response Headers
   - Limitation: Browser security prevents direct access
   - Workaround: Use Electron API for network interception
   - Documented: Yes, in STATUS.md
   - Priority: Medium (affects 5-10% of signatures)

2. Favicon Hash Collection
   - Limitation: Requires image buffer for SHA-256 hashing
   - Workaround: Optional feature (flag: enableFaviconHashing)
   - Documented: Yes
   - Priority: Low (enhancement feature)

3. Batch Detection Placeholder
   - Limitation: Requires full tab management
   - Workaround: Loop calling detect_technologies per tab
   - Documented: Yes
   - Priority: Low (Phase 2 enhancement)

4. Signature Coverage
   - Limitation: 87 signatures vs. Wappalyzer's 8000+
   - Workaround: Modular system for expansion
   - Path: Documented expansion path in STATUS.md
