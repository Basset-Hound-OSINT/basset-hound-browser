## Module 1: Mouse Movement Anonymization

**File:** `src/anonymity/mouse-movement.js` (340 LOC)

### Purpose
Generate realistic mouse movement curves that mimic human behavior, not straight lines.

### Features

#### Key Capabilities
- Bézier curve path generation (smooth, curved movements)
- Variable speed profiles (acceleration → constant → deceleration)
- Hesitation points (5-20% chance of mid-movement pauses)
- Micro-movements and random jitter (±2-5px)
- Overshooting and correction patterns
- Realistic movement duration calculation (50-300 px/s)

#### Implementation Highlights
- **Bézier Curves:** Cubic Bézier curves prevent straight-line detection
- **Speed Profile:** Non-uniform acceleration using ease-in/ease-out easing
- **Hesitation:** Probabilistic pauses at random points along path
- **Jitter:** Subtle micro-movements prevent perfect smoothness
- **Overshooting:** Cursor overshoots target, then corrects (human-like)
