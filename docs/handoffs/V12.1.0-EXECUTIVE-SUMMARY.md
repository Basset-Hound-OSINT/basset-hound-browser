## What's New in v12.1.0

### 6 Major Features (All Complete & Tested)

1. **Session Coherence Validation** - 5-layer real-time detection system
   - Validates IP/Network, TLS/HTTP, Device, Behavioral, and Session consistency
   - <5ms validation time (target met)
   - 145 tests, 100% passing

2. **Behavioral Coherence Scoring** - 4-dimensional anomaly detection
   - Analyzes mouse, typing, scrolling, and timing behavior
   - <500ms analysis window (target met)
   - 115+ tests, 100% passing

3. **Evidence Packaging & Chain of Custody** - Forensic-grade export
   - ISO 27037, NIST, ACPO, RFC 3161 compliant
   - 5 export formats (JSON, XML, court-ready, analysis, ZIP)
   - 85 tests, 100% passing

4. **Technology Fingerprinting & Detection** - Multi-method detection
   - Detects 200+ web technologies with 95%+ accuracy
   - 3 detection methods (signature, behavioral, heuristic)
   - 60+ tests, 95%+ accuracy

5. **Platform Integrations** - Enterprise system integration
   - Splunk, ELK, SIEM, and webhook support
   - Real-time event streaming and batch export
   - 40+ integration tests, 100% passing

6. **Compression Framework** - Intelligent bandwidth optimization
   - 70-93% reduction on large payloads
   - <5ms compression overhead
   - Transparent, automatic operation

✅ Path traversal prevention

✅ Input validation framework
