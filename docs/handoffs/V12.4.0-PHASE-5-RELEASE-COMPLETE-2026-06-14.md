## Deployment Instructions

### Prerequisites
- Node.js 18.x LTS
- npm 8+
- Git access
- Docker (optional, for containerized deployment)

### Standard Deployment

```bash
# 1. Update to v12.4.0
npm install basset-hound-browser@12.4.0

# 2. Verify tests pass (optional)
npm test -- tests/anonymity/

# 3. Start browser
npm start

# 4. Verify anonymity available
curl http://localhost:8765/status
# Should show: anonymity features available
```

### Docker Deployment

```bash
# Build with v12.4.0
docker build -t basset-hound-browser:12.4.0 .

# Run container
docker run -p 8765:8765 basset-hound-browser:12.4.0
```

### Verification Commands

```bash
# Run all anonymity tests
npm test -- tests/anonymity/

# Check specific profiles
node -e "
  const APM = require('./src/anonymity/anonymity-profile-manager');
  const mgr = new APM();
  console.log('Available profiles:', mgr.getAvailableProfiles().length);
  mgr.setProfile('iPhone 15 Pro');
  console.log('iPhone 15 Pro set:', mgr.getActiveProfile());
"

# Quick health check
node tests/anonymity/quick-health-check.js
```
