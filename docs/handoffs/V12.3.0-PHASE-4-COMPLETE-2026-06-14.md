#### 1.1 Metrics Collector (`src/infrastructure/metrics-collector.js`)
- **LOC:** 290
- **Purpose:** In-process metrics collection for application performance
- **Features:**
  - Request tracking (total, success, error, in-progress)
  - Latency metrics with percentiles (min, max, avg, P50, P95, P99)
  - Throughput calculation (msg/sec, bytes/sec)
  - Memory monitoring (heap usage, RSS, percentage)
  - Error tracking by type with recent history
  - Connection tracking (active, total, closed)
  - Custom metrics support (gauge, counter, histogram)
  - Sliding window calculations
  - Metrics snapshot and category-based queries

**Key Capabilities:**
```javascript
collector.recordRequestStart(requestId);
collector.recordRequestEnd(requestId, startTime, success, bytesTransferred);
collector.recordConnection(connected);
collector.recordError(errorType, message);
collector.recordCustomMetric(name, value, type);
collector.getMetrics(); // Full snapshot
```
