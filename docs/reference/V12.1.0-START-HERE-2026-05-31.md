## What's New in v12.1.0?

### 4 Major Features

**1. 🔍 Technology Detection**
- Identify 50+ web technologies (frameworks, CMSs, servers)
- Multiple detection methods (HTTP headers, HTML patterns, CSS)
- 95%+ accuracy with version detection
- [Learn More →](V12.1.0-FEATURES-INDEX-2026-05-31.md#technology-detection-module)

**2. ⚖️ Forensic Evidence Export**
- Court-ready evidence packages
- Chain of custody documentation
- Cryptographic verification
- SWGDE-compliant guidelines
- [Learn More →](V12.1.0-FEATURES-INDEX-2026-05-31.md#forensic-evidence-export)

**3. 🔗 Platform Integrations**
- Export to Shodan, Maltego, MISP, Censys, STIX/TAXII
- Webhook support for real-time notifications
- 5-minute setup for each platform
- [Learn More →](V12.1.0-FEATURES-INDEX-2026-05-31.md#platform-integrations)

**4. ⚡ Advanced JavaScript**
- Sandboxed script execution
- 10+ pre-built extraction templates
- React/Vue/Angular state extraction
- Safe payload testing
- [Learn More →](V12.1.0-FEATURES-INDEX-2026-05-31.md#advanced-javascript-execution)
