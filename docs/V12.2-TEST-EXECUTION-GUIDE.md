## Troubleshooting

### Tests Timing Out
```bash
# Increase timeout to 180 seconds
npm test -- v12.2-comprehensive-integration --testTimeout=180000

# Or for specific test
npm test -- v12.2-comprehensive-integration \
  --testNamePattern="specific-test" \
  --testTimeout=180000
```

### Memory Issues
```bash
# Increase Node.js heap size
NODE_OPTIONS="--max-old-space-size=4096" npm test -- v12.2-comprehensive-integration
```

### Specific Test Failures
```bash
# Debug single failing test
npm test -- v12.2-comprehensive-integration \
  --testNamePattern="exact-test-name" \
  --verbose

# Get stack trace
npm test -- v12.2-comprehensive-integration --bail
```

### Database/File Issues
```bash
# Clear test results directory
rm -rf tests/results/
mkdir -p tests/results/

# Run tests again
npm test -- v12.2-comprehensive-integration
```
