## Feature Quick Reference

### Feature 1: Compression (5 commands)

**Basic Usage:**
```javascript
// Auto-select best algorithm
const result = await client.send({
  command: 'compress',
  params: {
    data: Buffer.from(largeData).toString('base64'),
    contentType: 'application/json'
  }
});
console.log(`Ratio: ${result.ratio}, Algorithm: ${result.algorithm}`);
```

**Commands:**
1. `compress` - Auto-compress with best algorithm
2. `compressMultiple` - Test all algorithms
3. `getCompressionMetrics` - Get per-algorithm stats
4. `getCompressionStatistics` - Full statistics
5. `resetCompressionMetrics` - Reset tracking

**Performance:** <10ms, 0.25-0.35 ratio

### Feature 2: Orchestration (9 commands)

**Basic Usage:**
```javascript
// Register agent
await client.send({
  command: 'registerAgent',
  params: {
    agentId: 'agent-1',
    capabilities: ['work'],
    maxConcurrentTasks: 5
  }
});

// Submit task
const taskId = await client.send({
  command: 'submitTask',
  params: {
    task: { type: 'work', requiredCapabilities: ['work'] },
    priority: 'high'
  }
});

// Complete task
await client.send({
  command: 'completeTask',
  params: { taskId, result: { status: 'ok' } }
});
```

**Commands:**
1. `registerAgent` - Register new agent
2. `unregisterAgent` - Remove agent
3. `submitTask` - Queue task
4. `completeTask` - Mark as done
5. `failTask` - Mark as failed
6. `getTaskStatus` - Check status
7. `getAgentStatus` - Agent health
8. `getAllAgentsStatus` - All agents
9. `getOrchestratorStatistics` - Stats

**Performance:** 100+ tasks/sec, <5ms per op

## Common Tasks

### Task: Compress Large JSON

```javascript
const data = { largeObject: 'data' };
const compressed = await client.send({
  command: 'compress',
  params: {
    data: Buffer.from(JSON.stringify(data)).toString('base64'),
    contentType: 'application/json'
  }
});
console.log(`Compressed ${compressed.originalSize} to ${compressed.compressedSize} bytes`);
```

### Task: Run 1000 Tasks in Parallel

```javascript
// Register 10 agents
for (let i = 0; i < 10; i++) {
  await client.send({
    command: 'registerAgent',
    params: {
      agentId: `agent-${i}`,
      capabilities: ['work'],
      maxConcurrentTasks: 20
    }
  });
}

// Submit 1000 tasks
const taskIds = [];
for (let i = 0; i < 1000; i++) {
  const taskId = await client.send({
    command: 'submitTask',
    params: {
      task: { type: 'work', requiredCapabilities: ['work'] },
      priority: i < 100 ? 'high' : 'normal'
    }
  });
  taskIds.push(taskId);
}

// Complete all tasks
for (const taskId of taskIds) {
  await client.send({
    command: 'completeTask',
    params: { taskId, result: { ok: true } }
  });
}

// Get stats
const stats = await client.send({
  command: 'getOrchestratorStatistics',
  params: {}
});
console.log(`Completed ${stats.statistics.tasksCompleted} tasks`);
```

## Troubleshooting

### "Command not found"
- Ensure `registerV12_9_0Commands` is called in server init
- Verify integration file is loaded correctly

### "High compression latency"
- Use `getCompressionMetrics` to check by algorithm
- Switch to `gzip` if latency matters more than ratio

### "Tasks not executing"
- Check `getAllAgentsStatus` - are agents registered?
- Verify task `requiredCapabilities` match agent capabilities
- Check `getQueueStats` - is queue backing up?

### "Memory growing"
- Call `resetV12_9_0` to clear metrics
- Archive old forensic artifacts
- Reduce `maxArtifacts` if needed
