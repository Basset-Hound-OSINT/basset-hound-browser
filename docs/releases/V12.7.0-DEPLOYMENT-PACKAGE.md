## INSTALLATION AND DEPLOYMENT

### Quick Start (Development)

```bash
# 1. Update code
git pull origin main

# 2. Install dependencies
npm install

# 3. Run tests
npm test

# 4. Start server
npm start
```

### Docker Deployment (Production)

```bash
# 1. Build image
docker build -t basset-hound-browser:12.7.0 .

# 2. Run container
docker run -d \
  --name basset-hound-browser \
  --network basset-hound-browser \
  -p 8765:8765 \
  -e NODE_ENV=production \
  basset-hound-browser:12.7.0

# 3. Verify
curl http://localhost:8765/health
```

### Quick Reference

**Problem: Can't connect to WebSocket**
- Verify port 8765 is open: `netstat -tlnp | grep 8765`
- Check firewall: `sudo iptables -L | grep 8765`
- Review logs: `docker logs basset-hound-prod`

**Problem: Credentials not generating**
- Verify secret is Base32 encoded
- Check crypto support in Node.js version
- Review: `docs/API-REFERENCE-v12.7.0.md#generate_totp`

**Problem: Session snapshots failing**
- Check disk space for snapshot storage
- Verify IndexedDB is accessible
- Review: `docs/API-REFERENCE-v12.7.0.md#snapshot_session`

**Problem: Evasion not working**
- Verify behavioral profile is set
- Check detection service connectivity
- Review: `docs/API-REFERENCE-v12.7.0.md#test_detection_vectors`

**Problem: Metrics not available**
- Verify metrics collection enabled
- Check memory usage (metrics require buffering)
- Review: `docs/API-REFERENCE-v12.7.0.md#get_metrics`
