## DEPLOYMENT STEPS

### Phase 1: Canary Deployment (10% Traffic)

**Duration:** 5 minutes  
**Risk:** Very Low  
**Rollback:** Immediate

#### Step 1.1: Start Single Canary Instance

```bash
# Build v12.7.0 Docker image (if not pre-built)
docker build -t basset-hound-browser:12.7.0 .

# Create canary container on separate port
docker run -d \
  --name basset-hound-canary \
  --network basset-hound-browser \
  -p 8766:8765 \
  -e NODE_ENV=production \
  -e LOG_LEVEL=info \
  basset-hound-browser:12.7.0

# Wait for startup
sleep 4

# Verify canary is healthy
curl http://localhost:8766/health
```

   ```bash
   docker network create basset-hound-browser --driver bridge
   ```

2. **Port Availability:** Verify ports 8765-8770 are available (or adjust as needed)

3. **Configuration:** Ensure environment variables match production standards:
   ```bash
   NODE_ENV=production
   LOG_LEVEL=warn
   PORT=8765
   ```
