# v12.7.0 WebSocket Commands - Quick Reference

## Credentials (TOTP/HOTP)

### Generate TOTP Token
```javascript
{
  "command": "generate_totp",
  "secret": "GEZDGNBVGY3TQOJQ",
  "algorithm": "SHA1",           // Optional: SHA1 (default), SHA256, SHA512
  "window": 30,                  // Optional: seconds (default 30)
  "digits": 6                    // Optional: 6-8 (default 6)
}
// Response: {success, token, expiresAt, validFor, timeRemaining, digits, algorithm, window}
```

### Validate TOTP Token
```javascript
{
  "command": "validate_totp",
  "secret": "GEZDGNBVGY3TQOJQ",
  "token": "123456",
  "window": 1,                   // Optional: drift tolerance (default 1)
  "algorithm": "SHA1"            // Optional: (default SHA1)
}
// Response: {success, valid, message, token, driftWindow}
```

### Generate HOTP Token
```javascript
{
  "command": "generate_hotp",
  "secret": "GEZDGNBVGY3TQOJQ",
  "counter": 0,                  // Optional: counter value (default 0)
  "algorithm": "SHA1",           // Optional: (default SHA1)
  "digits": 6                    // Optional: 6-8 (default 6)
}
// Response: {success, token, counter, nextCounter, algorithm, digits}
```

### Validate HOTP Token
```javascript
{
  "command": "validate_hotp",
  "secret": "GEZDGNBVGY3TQOJQ",
  "token": "123456",
  "counter": 0,                  // Optional: current counter (default 0)
  "lookahead": 0                 // Optional: lookahead windows 0-100 (default 0)
}
// Response: {success, valid, counter, message, lookaheadUsed}
```

### Resync HOTP Counter
```javascript
{
  "command": "resync_hotp",
  "secret": "GEZDGNBVGY3TQOJQ",
  "correctCounter": 42,          // Required: known correct counter
  "algorithm": "SHA1"            // Optional: (default SHA1)
}
// Response: {success, counter, message}
```

### Get TOTP Information
```javascript
{
  "command": "get_totp_info",
  "secret": "GEZDGNBVGY3TQOJQ",
  "algorithm": "SHA1",           // Optional: (default SHA1)
  "timeWindow": 30               // Optional: seconds (default 30)
}
// Response: {success, timeRemaining, currentCounter, nextTokenStartsAt, windowSize, algorithm}
```

## Session Persistence

### Save Session State
```javascript
{
  "command": "save_session_state",
  "profileId": "profile1",       // Optional: profile ID (default 'default')
  "includeDOM": true,            // Optional: include DOM state (default true)
  "compress": true,              // Optional: compress state (default true)
  "metadata": {                  // Optional: custom metadata
    "label": "Before login"
  }
}
// Response: {success, sessionId, compressed, compressionRatio, sizeBytes, savedAt, url, title}
```

### Restore Session State
```javascript
{
  "command": "restore_session_state",
  "sessionId": "session-profile1-1-1718400000000",  // Required
  "validateFirst": true,         // Optional: validate before restore (default true)
  "skipDOM": false               // Optional: skip DOM restoration (default false)
}
// Response: {success, restored{}, failed{}, warnings[], errors[], restoreTime, sessionId, url}
```

### List Saved Sessions
```javascript
{
  "command": "list_saved_sessions",
  "profileId": "profile1",       // Optional: filter by profile
  "limit": 50,                   // Optional: max results (default 50)
  "offset": 0                    // Optional: pagination offset (default 0)
}
// Response: {success, sessions[], total, limit, offset, hasMore}
```

### Delete Session State
```javascript
{
  "command": "delete_session_state",
  "sessionId": "session-profile1-1-1718400000000"  // Required
}
// Response: {success, message, deletedSessionId}
```

### Verify Session State
```javascript
{
  "command": "verify_session_state",
  "sessionId": "session-profile1-1-1718400000000"  // Required
}
// Response: {success, valid, issues[], severity, warning, sessionId, checked}
```

### Get Session Metadata
```javascript
{
  "command": "get_session_metadata",
  "sessionId": "session-profile1-1-1718400000000"  // Required
}
// Response: {success, metadata{}, state{url, title, cookies, localStorage, sessionStorage, capturedAt, profileId}, sessionId}
```

## Extended Evasion

### Configure TLS Evasion
```javascript
{
  "command": "configure_tls_evasion",
  "domain": "example.com",       // Required
  "tlsVersion": "TLS1.2",        // Required: TLS1.0, TLS1.1, TLS1.2, TLS1.3
  "profile": "realistic"         // Optional: conservative, realistic, aggressive
}
// Response: {success, version, minVersion, supportedVersions[], handshakeParams{}, domain, profile}
```

### Configure HTTP/2 Headers
```javascript
{
  "command": "configure_http2_headers",
  "profile": "default",          // Optional: any string
  "strategy": "realistic"        // Optional: conservative, realistic, aggressive
}
// Response: {success, strategy, pseudoHeaders[], regularHeaders[], profile}
```

### Enable Timing Randomization
```javascript
{
  "command": "enable_timing_randomization",
  "minDelay": 5,                 // Optional: min delay in ms (default 5)
  "maxDelay": 150,               // Optional: max delay in ms (default 150)
  "burstThreshold": 5,           // Optional: burst threshold (default 5)
  "strategies": {                // Optional: custom timing per request type
    "resource": 15,
    "xhr": 60,
    "fetch": 70,
    "navigation": 100,
    "form": 150
  }
}
// Response: {success, enabled, minDelay, maxDelay, burstThreshold, strategies}
```

### Obfuscate Network
```javascript
{
  "command": "obfuscate_network",
  "domain": "example.com",       // Required
  "dnsQueryPattern": "standard", // Optional: standard, staggered, batched
  "connectionPoolSize": 6,       // Optional: pool size (default 6)
  "obfuscationLevel": "medium"   // Optional: low, medium, high
}
// Response: {success, domain, dnsPattern, poolSize, obfuscationLevel, ephemeralPorts[], responseDelay}
```

### Set Evasion Coherence
```javascript
{
  "command": "set_evasion_coherence",
  "domain": "example.com",       // Required
  "tlsVersion": "TLS1.2",        // Optional: (default TLS1.2)
  "level": "medium"              // Optional: low, medium (default), high
}
// Response: {success, domain, tlsVersion, level, coherenceScore, consistencyCheck{}}
```

### Get Evasion Metrics
```javascript
{
  "command": "get_evasion_metrics",
  "detailed": false              // Optional: detailed metrics (default false)
}
// Response: {success, metrics{activeSince, totalRequests, evasionHits, successRate, ...}}
```

## Common Response Format

All commands follow a consistent response format:

### Success Response
```javascript
{
  "success": true,
  "data": { /* command-specific data */ },
  "timestamp": 1718400000000
}
```

### Error Response
```javascript
{
  "success": false,
  "error": "Error message describing what went wrong"
}
```

## Usage Examples

### Complete TOTP Flow
```javascript
// 1. Generate token
ws.send(JSON.stringify({
  id: 1,
  command: "generate_totp",
  secret: "GEZDGNBVGY3TQOJQ"
}));
// → {success: true, token: "123456", expiresAt: ..., validFor: 25000}

// 2. Validate token
ws.send(JSON.stringify({
  id: 2,
  command: "validate_totp",
  secret: "GEZDGNBVGY3TQOJQ",
  token: "123456"
}));
// → {success: true, valid: true, message: "Token is valid"}
```

### Complete Session Flow
```javascript
// 1. Save session
ws.send(JSON.stringify({
  id: 1,
  command: "save_session_state",
  profileId: "profile1"
}));
// → {success: true, sessionId: "session-profile1-1-...", compressed: true, sizeBytes: 45000}

// 2. Restore session
ws.send(JSON.stringify({
  id: 2,
  command: "restore_session_state",
  sessionId: "session-profile1-1-..."
}));
// → {success: true, restored: {cookies: 5, storage_items: 12}, warnings: []}

// 3. Clean up
ws.send(JSON.stringify({
  id: 3,
  command: "delete_session_state",
  sessionId: "session-profile1-1-..."
}));
// → {success: true, message: "Session deleted", deletedSessionId: "..."}
```

## Performance Tips

1. **Credentials**: Generate and validate in parallel for MFA systems
2. **Session Persistence**: Save state before risky operations (login, checkout)
3. **Evasion**: Configure once per session, not per request
4. **Monitoring**: Sample metrics at 5-10 second intervals, not continuously

## Error Handling

Always check `response.success` before using `response.data`:

```javascript
const response = JSON.parse(message);
if (response.success) {
  console.log("Success:", response.data);
} else {
  console.error("Error:", response.error);
}
```
