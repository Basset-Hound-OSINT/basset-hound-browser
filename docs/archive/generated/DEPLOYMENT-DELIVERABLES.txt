QUICK REFERENCE COMMANDS
================================================================================

Pre-Deployment:
```bash
npm audit                              # Security audit
npm run lint                           # Code quality
npm run test:unit                      # Unit tests
npm run test:integration               # Integration tests
docker build -t basset-hound-browser:12.8.0 .  # Build image
```

Staging Deployment:
```bash
docker-compose -f docker-compose.staging.yml up -d  # Deploy
curl -f http://staging:8765/health                  # Health check
docker logs -f basset-hound-staging --tail 100      # Logs
docker stats basset-hound-staging --no-stream       # Stats
```

Production Deployment:
```bash
docker pull ${REGISTRY}/basset-hound-browser:12.8.0        # Pull image
docker run -d --name basset-hound-browser-prod \           # Start container
  --memory 2g --cpus 2 \
  -e NODE_ENV=production \
  -e RATE_LIMIT_ENABLED=true \
  -p 8765:8765 \
  ${REGISTRY}/basset-hound-browser:12.8.0
```

Health Check:
```bash
curl -f http://localhost:8765/health              # Health endpoint
docker stats basset-hound-browser-prod --no-stream # Resource usage
docker logs -f basset-hound-browser-prod          # Logs
```

Rollback:
```bash
docker stop basset-hound-browser-prod
docker run -d --name basset-hound-browser-prod \
  -p 8765:8765 \
  ${REGISTRY}/basset-hound-browser:12.7.0
curl -f http://localhost:8765/health              # Verify
```
