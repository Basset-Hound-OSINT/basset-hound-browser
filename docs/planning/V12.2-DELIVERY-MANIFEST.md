## Quick Start

### Run All Tests
```bash
npm test -- v12.2-comprehensive-integration
npm test -- v12.2-comprehensive-scenarios
```

### Run by Category
```bash
npm test -- v12.2-comprehensive-integration --testNamePattern="Feature Integration"
npm test -- v12.2-comprehensive-integration --testNamePattern="Performance Baselines"
```

### Run by Scenario Group
```bash
npm test -- v12.2-comprehensive-scenarios --testNamePattern="Competitive Intelligence"
npm test -- v12.2-comprehensive-scenarios --testNamePattern="Campaign Orchestration"
```

### Advanced Options
```bash
npm test -- v12.2-comprehensive-integration --verbose
npm test -- v12.2-comprehensive-integration --coverage
npm test -- v12.2-comprehensive-scenarios --testTimeout=180000
npm test -- v12.2-comprehensive-integration --bail
npm test -- v12.2-comprehensive-integration --watch
```
