## API Overview

### 30 WebSocket Commands

**Monitor Management (7):**
- `add_competitor_monitor` - Create new monitor
- `list_competitor_monitors` - List with filtering
- `get_competitor_monitor` - Get details
- `update_competitor_monitor` - Modify config
- `pause_competitor_monitor` - Pause monitoring
- `resume_competitor_monitor` - Resume monitoring
- `remove_competitor_monitor` - Delete monitor

**Change Tracking (4):**
- `check_competitor_monitor` - Trigger manual check
- `get_competitor_changes` - Retrieve change history
- `get_competitor_snapshots` - Get snapshot history
- `get_competitor_stats` - Get monitor statistics

**Service Control (7):**
- `start_competitor_monitoring` - Start service
- `stop_competitor_monitoring` - Stop service
- `pause_competitor_monitoring` - Pause service
- `resume_competitor_monitoring` - Resume service
- `get_competitor_monitoring_status` - Get service status
- `get_competitor_monitoring_stats` - Get global stats
- `run_competitor_monitoring_checks` - Run immediate checks

**Alert Configuration (1):**
- `configure_competitor_alerts` - Configure alerts

**Data Management (4):**
- `export_competitor_monitoring_data` - Export all data
- `import_competitor_monitoring_config` - Import config
- `cleanup_competitor_monitoring_data` - Cleanup old data
- `clear_all_competitor_monitors` - Clear all monitors

## Change Detection Capabilities

### Detection Types

1. **Content Changes** (Text-based)
   - SHA-256 hashing for comparison
   - Word count differences
   - Length changes
   - Change percentage calculation

2. **Structure Changes** (DOM-based)
   - Element count tracking
   - Heading changes
   - Form structure changes
   - Section changes

3. **Technology Changes** (Framework detection)
   - Framework changes: React, Vue, Angular, Next.js, Django, Rails
   - Server technology: Apache, Nginx, IIS, Node.js, Cloudflare
   - Libraries: jQuery, Bootstrap, Tailwind, Axios, Lodash
   - CMS: WordPress, Joomla, Drupal, Magento, Shopify
   - Version tracking

4. **Performance Changes** (Metrics-based)
   - Load time changes >10%
   - Resource count changes
   - DOM size changes
   - Threshold-based alerting

5. **Status Changes** (HTTP-based)
   - Status code changes
   - Availability changes
   - Response header changes
