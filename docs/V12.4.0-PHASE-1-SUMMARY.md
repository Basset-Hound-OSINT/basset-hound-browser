## Usage Example

```javascript
// Set iPhone profile
{
  command: 'set_anonymity_profile',
  params: { profileName: 'iPhone 15 Pro' }
}

// Set custom values
{
  command: 'set_anonymity_custom',
  params: {
    properties: {
      hardwareConcurrency: 8,
      deviceMemory: 16,
      screenWidth: 1920
    }
  }
}

// Get random mobile device
```
