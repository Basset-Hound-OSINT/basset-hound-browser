## Video 4: Proxy Setup & Rotation (12 minutes)

### TARGET AUDIENCE
- Advanced users
- Large-scale monitoring needs
- Users in restricted networks

### PRODUCTION SPECS
- **Duration:** 12 minutes
- **Format:** Screen recording + voiceover + diagrams
- **Resolution:** 1920x1080 @ 30fps
- **Narration:** Technical, precise, professional

### SCRIPT

#### SECTION 1: Why Proxies Matter (0:00-1:30)
**VISUALS:** Diagram showing request flow with and without proxies

**NARRATION:**
"When monitoring at scale, proxies are essential. They mask your identity, distribute requests across multiple IP addresses, and allow you to monitor geo-specific content. Some websites block automation, others restrict request rates. Proxies help you stay under the radar while maintaining reliable monitoring."

#### SECTION 2: Proxy Configuration (1:30-5:00)
**VISUALS:** Screen recording of proxy settings interface

**NARRATION:**
"In your monitor settings, find the Proxy section. You can configure HTTP, HTTPS, or SOCKS proxies. Enter the proxy address and port. If authentication is required, add your username and password. Basset Hound Browser supports proxy rotation—automatically switching between proxies for each request. This prevents rate limiting and IP-based detection."

**PRODUCTION NOTES:**
- Show HTTP proxy example
- Show SOCKS5 proxy example
- Highlight authentication fields
- Display test connection button

#### SECTION 3: Proxy Rotation Strategies (5:00-8:30)
**VISUALS:** Diagrams and screen recordings of rotation modes

**NARRATION:**
"We offer three rotation strategies. Round-robin cycles through your proxy list sequentially—best for load distribution. Random picks a random proxy for each request—best for evasion. Sticky maintains one proxy per session—best for consistency and reliability. Choose based on your needs. For most competitive intelligence, round-robin or random work best."

**PRODUCTION NOTES:**
- Show each strategy with diagrams
- Display proxy switching in logs
- Show performance metrics for each

#### SECTION 4: Residential Proxies (8:30-11:00)
**VISUALS:** Explanation of residential vs datacenter proxies

**NARRATION:**
"Residential proxies are the gold standard. They route traffic through real residential ISPs, making requests appear like normal user behavior. They're essential for highly protected sites. Basset Hound Browser integrates with major residential proxy providers. Simply enter your credentials, and we handle the rest."

**PRODUCTION NOTES:**
- Show comparison chart: datacenter vs residential
- Display success rates with and without residential
- Show integration setup for popular providers

#### SECTION 5: Testing and Monitoring (11:00-12:00)
**VISUALS:** Screen recording of proxy health checks and logs

**NARRATION:**
"Always test your proxy configuration. Our built-in proxy tester checks connectivity and performance. Monitor proxy health in real-time. If a proxy fails, the system automatically switches to the next one. You can view detailed logs of every proxy request to troubleshoot issues quickly."

**PRODUCTION NOTES:**
- Show test results interface
- Display health check metrics
- Show automatic failover in action
