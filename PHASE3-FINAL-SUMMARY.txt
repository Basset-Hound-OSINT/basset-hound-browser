ARCHITECTURE OVERVIEW
================================================================================

BaseManager (580 LOC):
  • Lifecycle Management: Constructor → Initialize → Validate → Cleanup
  • 7 Lifecycle States: UNINITIALIZED, INITIALIZING, INITIALIZED, VALIDATING, 
                       READY, ERROR, CLEANUP
  • Unified Logging: Automatic, consistent log formatting
  • Error Handling: Automatic error tracking with recovery
  • Performance Metrics: Operation counting, timing, error tracking
  • Safe Execution: safeExecute() with timeout protection
  • Health Monitoring: isHealthy(), isReady() checks
  • Status Reporting: Comprehensive getStatus() method

ManagerRegistry (576 LOC):
  • Manager Registration: register/unregister/lookup/list
  • Coordinated Initialization: initializeAll() with ordering support
  • Coordinated Validation: validateAll() for all managers
  • Graceful Shutdown: cleanupAll() in reverse order (LIFO)
  • Health Monitoring: Aggregate health status, individual checks
  • Error Recovery: Configurable continueOnError option
  • Statistics Tracking: Registration, initialization, validation events

USAGE EXAMPLE
================================================================================

// 1. Create a manager extending BaseManager
const { BaseManager } = require('./src/managers');

class CustomManager extends BaseManager {
  constructor() {
    super('CustomManager');
    this.customData = null;
  }
  
  async _baseInitialize() {
    // Custom initialization logic
    this.customData = { initialized: true };
  }
  
  async _baseValidate() {
    // Custom validation logic
    return { valid: !!this.customData };
  }
}

// 2. Create registry and register managers
const { ManagerRegistry } = require('./src/managers');

const registry = new ManagerRegistry({
  initializationOrder: ['proxy', 'session', 'headers']
});

registry.register('custom', new CustomManager());

// 3. Initialize all managers
const result = await registry.initializeAll();

// 4. Monitor health
const health = registry.getHealthStatus();
console.log('Health:', health.overallHealth);

// 5. Cleanup on shutdown
await registry.cleanupAll();
